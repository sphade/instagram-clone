## Instagram

this is a project that aims to replicate some of the simple functionalities of the official Instagram Web Application

### `View Live Project`

[click to view live project]()



### `Functionalities`

Auth users are able to make post (pictures) and include captions and users (including the one who made the post) are able to comment on the post

### `The Stack`

- REACT
- REDUX
- FIREBASE
- FIRESTORE
- GOOGLE AUTH
